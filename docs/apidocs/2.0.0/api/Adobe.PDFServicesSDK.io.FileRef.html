Class FileRef

This class represents either a local file or a URL-based web page. It is typically used by IOperation<T> implementations which accept or return files.

If a FileRef instance is created by this SDK and it refers to a temporary file location, calling any of the overloaded saveAs methods will delete the temporary file.
